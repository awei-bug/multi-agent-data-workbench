# 多 Agent 协同数据分析创作工作台 - 项目总览

## 1. 项目定位

这是一个面向“数据分析 + 多 Agent 协同创作”场景的工作台项目。

目标是提供一套可运行的工作台，用于串联以下主流程：

1. 数据源接入
2. 分析需求输入
3. 意图解析
4. 多 Agent 任务流执行
5. 数据清洗与指标计算
6. 可视化图表生成
7. 报告编辑与导出
8. 历史任务复盘与模板复用

当前项目已经落地为一个可运行系统，包含前端工作台和后端 API。

## 3. 项目结构

### 根目录关键内容

- `src/`
  前端源码目录
- `backend_py/`
  Python 后端源码目录
- `scripts/`
  启动脚本和辅助脚本
- `docs/`
  设计文档、计划文档
- `dist/`
  前端生产构建产物
- `tests/`
  E2E 或验证脚本
- `package.json`
  前端脚本与依赖定义
- `requirements.txt`
  Python 依赖定义
- `多Agent协同数据分析.md`
  原始需求/说明文档

### `src/` 前端主要模块

- `components/`
  通用组件与工作台模块组件
- `pages/`
  页面级组件
- `context/`
  全局上下文
- `mock/`
  前端原型数据与模拟任务仓库
- `server/`
  前端侧会话传输与 API 桥接逻辑
- `styles/`
  全局样式与主题样式
- `types/`
  类型定义

### `backend_py/` 后端主要职责

- 提供工作台任务会话接口
- 提供模板、历史任务、工作流动作接口
- 对接 Agnes 模型接口
- 连接 PostgreSQL

## 5. 前后端运行方式

### 当前稳定运行方式

这个项目当前稳定运行方式不是 Vite 的 `dev` 模式，而是：

1. 前端先构建
2. 再通过 `vite preview` 提供预览服务
3. 后端通过 Python 脚本启动

### 前端命令

构建：

```bash
npm run build
```

预览：

```bash
npm run preview -- --host 127.0.0.1 --port 4173
```

### 后端命令

```bash
python scripts/start-workspace-api.py
```

实际运行依赖以下环境变量：

- `AGNES_API_KEY`
- `WORKSPACE_DATABASE_URL`
- `WORKSPACE_API_PORT`

优先使用：

1. `npm run build`
2. `npm run preview -- --host 127.0.0.1 --port 4173`
3. `python scripts/start-workspace-api.py`

### 调试重点

如果出现“地址变了但页面内容不切换”，优先检查：

1. `src/` 目录是否又出现残留 `.js` 编译产物
2. 当前访问的是不是旧的 `dev` 端口
3. 是否仍在使用不稳定的 `5174` 开发服务
